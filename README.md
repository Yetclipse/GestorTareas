Gestor de Tareas con Flask

Aplicación web desarrollada en Python utilizando Flask y SQLite para la gestión de tareas.

Funcionalidades

Crear tareas

Marcar tareas como completadas

Eliminar tareas

Añadir categoría

Añadir fecha límite

Interfaz personalizada con Bootswatch

Tecnologías

Python

Flask

SQLAlchemy

SQLite

Bootstrap (Sketchy)

HTML & CSS

Instalación

Clonar el repositorio:

git clone https://github.com/Yetclipse/GestorTareas.git

Crear entorno virtual:

python -m venv venv

Activar entorno virtual:

Windows:
venv\Scripts\activate

Linux/Mac:
source venv/bin/activate

Instalar dependencias:

pip install -r requirements.txt

Ejecutar:

python main.py
